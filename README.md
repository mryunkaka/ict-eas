# ICT EAS

Baseline project Sistem Informasi ICT EAS berbasis Laravel 13.

## Modul inti
- Permintaan Fasilitas ICT
- Permohonan Email Account
- Permohonan Perbaikan ICT
- Berita Acara Kerusakan ICT
- Master Asset
- Pengajuan Project
- Inventory / stok
- Approval Center per role
- Report dan export Excel/PDF
- Tools Management User
- Tools Ping Server
- Disposal / transfer workflow asset
- CCTV maintenance log
